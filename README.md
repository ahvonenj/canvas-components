# Canvas Components

Allows for creation of inputs and other components in a HTML 5 canvas.
Aiming for easy extendability and ease-of-use.

## Supported / fully implemented components
- [ ] Buttons
- [ ] Checkboxes
- [ ] Panels for grouping components
- [ ] Radio buttons
- [ ] Text inputs
- [ ] Labels
- [ ] Range sliders
- [ ] Progress bars
